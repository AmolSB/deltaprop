What is it?

DeltaProp

Browse internet in an organized way

DeltaProp lets you save your bookmakrs across all devices. Just create a new account, create a collection to categorize your link and save your links

What's new in that?

DeltaProp also lets you see public collections (group of links), created by people like you, for you.

The idea behind it

Suppose you are a Beatles fan, browsing through internet, checking all the links available to get some info. There are chances that there is a page which contains very good info about Beatles but is't on page 6 or 7 or 8 (or maybe it's not event ranked in top 10)

Now how would you feel if I tell you you can see all sites which contains good info about The Beatles in one place?

That's where DeltaProp comes into play. A user goes through all these trouble and create a collection to save all those links. If he choses to make it public, anyone can just go through that public collection to get the hidden gems on internet.
